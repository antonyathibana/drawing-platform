## To Run the Application:

1. **Open Terminal and navigate to the project:**
   ```bash
   cd "/Users/antonyadith/Desktop/draw platform"
   ```

2. **Start the server:**
   ```bash
   npm start
   ```
   Or for development with auto-restart:
   ```bash
   npm run dev
   ```

3. **Open your browser:**
   - Go to: http://localhost:3000
   - Open the same URL in another browser window/tab for testing collaboration

## Testing Steps:
1. Enter a room code (e.g., 123456) in both browser windows
2. Click "Join Room" in both windows
3. Draw in one window - see it appear in real-time in the other
4. Test "Clear Canvas" and "Back to Home" buttons
